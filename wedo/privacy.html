WEDO 개인정보처리방침

시행일: 2024-07-01 · 최종 수정일: 2026-07-15 · 적용 버전: v1.1.4 이상

이전 버전에는 해당 버전 출시 시점의 처리방침이 적용됩니다. 변경 이력은 본 페이지 하단을 확인하세요.

hoho studio(이하 "회사")는 「개인정보 보호법」 등 관련 법령을 준수하며, WEDO 이용자의 개인정보를 다음과 같이 처리합니다.

1. 처리하는 개인정보 항목

1.1 자동 수집 항목

항목수집 방법필수/선택
익명 앱 인스턴스 식별자, OS 버전, 앱 버전, 기기 모델앱 실행 시 자동 수집필수
IP 주소앱 실행 시 자동 수집필수
앱 사용 이벤트(화면 진입·기능 사용 등 익명 통계)앱 사용 시 자동 수집필수
크래시·오류 로그앱 오류 발생 시 자동 수집필수

1.2 이용자 직접 입력 항목

항목수집 방법필수/선택
할 일·체크리스트 텍스트이용자 직접 입력선택
그룹 이름, 그룹 멤버 초대 정보이용자가 그룹 생성·참여 시 입력선택
배경 사진이용자가 배경 꾸미기 기능 사용 시 앨범에서 선택선택

1.3 권한 기반 수집 항목

항목수집 방법필수/선택
위치정보(대략적 위치, 시/구 단위)날씨·AI 브리핑 기능 사용 시, 이용자 권한 허용 시선택
사진/미디어 접근 권한배경 사진 꾸미기 기능 사용 시, 이용자가 앨범 선택 시선택
권한 안내 — 위치·사진 권한은 각 기능을 사용할 때 안내되며, 거부해도 앱의 다른 기능은 정상 이용할 수 있습니다. 권한은 언제든 기기 설정에서 철회할 수 있습니다.

2. 수집·이용 목적

  • WEDO 서비스 제공(할 일·체크리스트 저장 및 잠금화면 위젯 표시)
  • 그룹 공유 기능 제공(그룹 참여자 간 할 일 공유·동기화)
  • 현재 위치 기반 날씨 정보 제공
  • AI 브리핑(뉴스·날씨·명언·추천 일정) 제공
  • 앱 오류 진단 및 서비스 품질 개선
  • 앱 이용 패턴 분석을 통한 사용성 개선 (익명 통계)
  • 중요 공지·업데이트 안내

3. 보관 및 이용 기간

  • 앱 내 로컬 데이터(할 일·설정): 이용자의 디바이스에 저장. 앱 삭제 시 함께 삭제됩니다.
  • 그룹 공유 데이터: 그룹이 유지되는 동안 서버에 보관. 이용자가 그룹을 나가거나 삭제하면 해당 그룹에서 이용자 정보가 제거되며, 그룹 자체가 삭제되면 관련 데이터가 함께 삭제됩니다.
  • 배경 사진: 이용자 디바이스에 로컬 저장되며 외부로 전송되지 않습니다.
  • 위치정보: 날씨·AI 브리핑 요청 시점에 사용되며 별도 저장되지 않습니다.
  • 크래시·오류 로그: 수집일로부터 90일 (Firebase Crashlytics 보관 정책)
  • 익명 사용 통계: 수집일로부터 14개월 (Firebase Analytics 보관 정책)
  • 법령상 보관 의무: 관계 법령에서 보관을 요구하는 경우 해당 기간 동안 보관

4. 제3자 제공

회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 이용자가 그룹 공유 기능을 사용하는 경우, 이용자가 입력한 할 일·체크리스트 텍스트는 이용자가 초대한 그룹 참여자에게 공유됩니다. 이는 이용자의 자발적 공유이며, 제3자 제공에 해당하지 않습니다.

5. 처리 위탁

수탁자위탁 업무국가
Google LLC (Google Play Services)앱 배포 및 실행 인프라미국
Google LLC (Firebase Analytics)익명 사용 통계 수집·분석미국
Google LLC (Firebase Crashlytics)크래시·오류 로그 수집·분석미국
Google LLC (AdMob)앱 내 광고 서빙미국
국외 이전 고지 — 위 미국 수탁자에게는 개인을 특정할 수 없는 익명화된 기술 정보(익명 앱 인스턴스 식별자·OS·기기 모델·앱 사용 이벤트·크래시 로그 등)가 전송됩니다. 모든 전송 데이터는 전송 구간 TLS 암호화가 적용됩니다. 이전 거부 시 앱 사용에 제한이 있을 수 있으며, 이 경우 앱을 삭제하여 수집을 중단할 수 있습니다.

6. 파기 절차 및 방법

  • 파기 절차: 이용 목적이 달성된 개인정보는 지체 없이 파기합니다.
  • 파기 방법:
    • 전자적 파일: 복구 불가능한 방법으로 삭제
    • 출력물: 분쇄 또는 소각

7. 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람·정정·삭제·처리 정지 요구
  • 위치·사진 등 권한 철회 (기기 설정에서 언제든 가능)
  • 그룹 나가기 또는 그룹 삭제 (앱 내에서 직접)
  • 앱 삭제를 통한 로컬 데이터 전체 파기
  • 기타 권리 행사 및 데이터 삭제 요청은 아래 연락처로 요청 가능

8. 개인정보 자동 수집 장치 및 광고

본 앱은 앱 운영을 위해 익명 앱 인스턴스 식별자(Firebase Installation ID)를 수집합니다. 이는 광고 ID(AAID)가 아닙니다.

광고 표시 — 본 앱은 Google AdMob을 통해 광고를 표시합니다. AdMob의 데이터 처리 방침은 Google의 광고 정책을 참조하세요. 광고 표시 자체를 원하지 않으시면 앱을 삭제하시면 됩니다.

9. 만 14세 미만 아동의 개인정보 처리

WEDO는 만 14세 미만 아동을 대상으로 하지 않습니다. 회사는 만 14세 미만 아동으로부터 고의로 개인정보를 수집하지 않으며, 만약 만 14세 미만 아동이 개인정보를 제공한 사실이 확인되면 즉시 해당 정보를 삭제합니다. 보호자께서 자녀가 개인정보를 제공한 사실을 알게 된 경우 아래 연락처로 알려주시면 즉시 필요한 조치를 취하겠습니다.
