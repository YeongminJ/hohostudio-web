키즈건강비서 지원

앱 사용 중 궁금한 점, 의견, 버그 제보를 한 곳에 모았어요. 답을 못 찾으시면 아래 이메일로 알려주세요.

💌 문의 이메일: kids@hohostudio.co.kr — 보통 1~2일 안에 답변드려요.

자주 묻는 질문

1. 자녀 건강 정보는 어디에 저장되나요?

기본적으로 모든 기록은 본인 기기 안에만 저장되며 외부 서버로 전송되지 않아요. 가입·로그인도 필수가 아니에요. v1.2부터 추가된 클라우드 백업을 직접 켜시면 기록이 암호화되어 백업 서버로 안전하게 전송됩니다(AES-256-GCM 서버측 암호화). 켜지 않으면 기존처럼 기기 안에만 머물러요.

2. 기기를 바꾸면 기록이 어떻게 되나요?

두 가지 방법이 있어요.

  • 클라우드 백업 (간편, v1.2 이상): 설정 → 클라우드 백업을 켜두면 자동으로 백업됩니다. 새 기기에서 같은 소셜 계정으로 로그인하시면 "복원하기"로 한 번에 가져올 수 있어요.
  • 파일 내보내기 (수동): 설정 → 백업 파일 관리에서 JSON 파일로 저장해두고, 새 기기에서 "가져오기"로 복원할 수 있어요. 계정 없이도 가능한 방식이에요.

3. 로그인은 어떤 방식으로 하나요?

로그인은 선택사항이에요 — 가입 없이 익명 모드로도 모든 핵심 기능을 쓸 수 있어요. 로그인하시려면 Apple · Google · Kakao · Naver 4종 소셜 로그인 중 편하신 걸로 가입하시면 돼요. 이메일·비밀번호 직접 입력 방식은 없어요. 로그인을 켜면 클라우드 백업·기기 간 동기화를 사용할 수 있어요.

4. 계정을 삭제하면 데이터가 어떻게 되나요?

v1.2.1 이상에서 앱 내 설정 → 계정 → 계정 삭제로 직접 삭제할 수 있어요. 삭제되는 항목은 로그인 계정 정보 + 클라우드 백업 데이터이며, 디바이스에 저장된 자녀 건강 기록은 유지됩니다. 다른 소셜 계정으로 다시 로그인하거나 익명 모드로 계속 사용하실 수 있어요. 로컬 기록까지 모두 지우려면 계정 삭제 후 앱을 추가로 삭제하시면 돼요. 자세한 내용은 데이터 삭제 요청 페이지에서 확인하실 수 있어요.

5. 클라우드 백업은 안전한가요?

백업 데이터는 백업 서버(Cloudflare 아시아 태평양 region)에 도달하기 전 전송 구간(TLS)과 저장 시 AES-256-GCM으로 이중 암호화됩니다. 암호화 키는 데이터와 분리된 별도 보관소에 저장되어, 본 앱 서비스 외에는 누구도 백업 내용을 평문으로 볼 수 없어요. 동의를 철회하시거나 백업 삭제를 실행하시면 암호문과 키가 즉시 함께 삭제됩니다.

6. 발열 관리 모드는 어떻게 시작되나요?

체온을 38℃ 이상으로 기록하면 자동으로 "발열 관리 모드"가 시작돼요. 그 모드 안에서 시간 흐름에 따른 체온 변화, 해열제 복용, 진료 방문이 자동으로 묶여요. 평열로 24시간 유지되면 자동으로 종료돼요. 수동으로 종료할 수도 있어요.

7. 예방접종이나 검진 일정이 우리 아이와 안 맞아요

앱이 보여주는 일정은 질병관리청 표준 국가예방접종(NIP) 일정과 보건복지부·국민건강보험공단의 영유아 건강검진 표준을 따른 일반 기준이에요. 의료진의 개별 진료 소견과 다를 수 있어요. 의문이 있으시면 다니시는 소아청소년과에 문의해주세요.

8. 성장 곡선 백분위가 정확한가요?

앱은 2017 한국 소아청소년 성장도표(질병관리청·대한소아과학회)의 LMS 계수를 기반으로 백분위를 계산해요. 일반적인 기준치이며, 의학적 진단을 대체하지 않아요. 정확한 평가는 영유아 건강검진을 통해 받아주세요.

9. 꽃가루 정보가 안 보이거나 다른 지역이 나와요

꽃가루 데이터는 기상청 보건기상지수(V4 API)를 사용해요. 위치 권한을 허용하시면 가까운 광역지자체 기준으로 표시돼요. 권한이 없거나 다른 지역을 보고 싶으시면 홈 화면 꽃가루 카드의 지역 선택 메뉴로 직접 바꿀 수 있어요.

10. 수면 사운드는 어떻게 사용하나요?

홈 화면의 수면 사운드 카드에서 7가지 자연음 중 하나를 선택해 재생할 수 있어요 (드라이기 · 심장 박동 · 귀뚜라미 · 숲 바람 · 잔잔한 파도 · 시냇물 · 빗소리). 자동 종료 타이머를 설정하면 정한 시간 후 자연스럽게 꺼지고, 잠금 화면이나 다른 앱 사용 중에도 계속 재생됩니다.

11. 알림이 안 와요

설정 → 알림 화면에서 예방접종·검진 리마인더를 켜두셨는지, 그리고 기기 설정의 알림 권한이 허용되어 있는지 확인해주세요.

12. 자녀 정보를 잘못 입력했어요

프로필 탭에서 자녀 카드를 누르면 편집 화면으로 들어가요. 이름·생년월일·성별·키·몸무게 모두 수정 가능해요. 자녀 삭제도 같은 화면에서 가능하고, 삭제하면 그 자녀의 모든 기록이 함께 사라져요.

13. 광고가 있나요? 인앱 결제가 있나요?

현재 광고와 인앱 결제 모두 없습니다. 향후 일부 부가 기능(예: 처방약 OCR, 양육자 공유 등)은 유료 옵션으로 도입될 수 있으며, 앱 운영을 위해 광고를 도입할 가능성도 있어요. 다만 핵심 기록 기능은 계속 무료로 유지할 계획이에요. 도입 시 사전 안내드릴게요.

14. 피드백이나 새 기능 제안은 어떻게 보내나요?

앱 안 "프로필 → 앱 정보" 화면에서 핑크색 "피드백을 보내주세요" 카드를 누르면 의견 폼이 열려요. 또는 kids@hohostudio.co.kr로 메일 주셔도 좋아요. 한 줄도 큰 힘이 돼요.
