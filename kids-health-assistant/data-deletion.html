키즈건강비서 데이터 삭제 요청

앱: 키즈건강비서 (Kids Health Assistant) · 개발자: hoho studio · 최종 수정일: 2026-06-24

키즈건강비서는 이용자가 입력한 자녀 건강 기록과 계정 정보를 언제든지 삭제할 수 있도록 다음 방법을 제공합니다. 삭제 요청 시 본 페이지에 명시된 데이터 유형이 영구 삭제되며, 복구할 수 없습니다.

1. 앱 내에서 직접 삭제 (즉시 적용)

1.1 로컬 데이터 (모든 기록)

앱을 기기에서 삭제(uninstall)하면, 기기에 저장된 모든 자녀 건강 기록(SQLite 데이터베이스)이 함께 영구 삭제됩니다.

1.2 클라우드 백업 데이터 (v1.2.0부터)

앱 내 설정 → 클라우드 백업 → 백업 데이터 삭제를 실행하면, Cloudflare 백업 서버에 보관된 암호문과 암호화 키가 즉시 함께 삭제됩니다.

1.3 계정 정보 (소셜 로그인 정보) — v1.2.1 이상

v1.2.1 이상에서 앱 내 설정 → 계정 → 계정 삭제를 실행하면, Firebase Authentication에 저장된 계정 정보(이메일·이름·사용자 ID)와 클라우드 백업 데이터가 함께 삭제됩니다. 본인 확인을 위해 같은 소셜 계정으로 다시 로그인하는 절차가 포함됩니다.

로컬에 저장된 자녀 건강 기록은 이 절차로 삭제되지 않습니다. 다른 소셜 계정으로 다시 로그인하거나 익명 모드로 계속 사용하실 수 있도록 디바이스 안 기록은 유지됩니다. 로컬 기록까지 모두 지우려면 1.1 절차(앱 삭제)를 함께 진행해주세요.

Apple로 로그인하신 분의 경우, 「Sign in with Apple」 정책에 따라 Apple 측의 인증 연결도 함께 해제됩니다. Google · Kakao · Naver 본 계정 자체는 해당 서비스에서 직접 관리해주세요(본 앱은 해당 서비스에서 받은 정보만 삭제할 수 있습니다).

v1.2.0 이하 버전을 사용 중이시면 앱 내 직접 삭제 기능이 없으므로 아래 「이메일로 삭제 요청」 절차를 이용해주세요.

2. 이메일로 삭제 요청

앱에 접근할 수 없거나 기기를 분실하신 경우 다음 이메일로 요청해주세요.

💌 요청 이메일: kids@hohostudio.co.kr

요청 시 이메일 본문에 다음을 포함해주세요:

  • 제목: 데이터 삭제 요청
  • 로그인에 사용한 이메일 주소 또는 소셜 로그인 제공자(Apple·Google·Kakao·Naver) 중 하나
  • 삭제 범위: ① 전체 계정 삭제, ② 클라우드 백업만 삭제, ③ 특정 자녀 프로필만 삭제 중 선택

본인 확인 절차 후 관계 법령(개인정보 보호법 시행령 제43조)에 따라 요청 접수일로부터 10일 이내에 처리하며, 가능한 빠르게 답변드립니다.

3. 삭제되는 데이터 유형

본 앱에 입력된 자녀의 건강정보는 「개인정보 보호법」 제23조에 따른 민감정보이며, 보호자 요청 시 우선 삭제됩니다.

전체 계정 삭제 요청 시 다음 데이터가 모두 영구 삭제됩니다.

데이터 유형저장 위치
자녀의 이름·생년월일·성별 이용자 기기 + 클라우드 백업(동의 시)
신장·체중·머리둘레·체온 등 건강 측정값 이용자 기기 + 클라우드 백업(동의 시)
수유·수면·배변·투약 기록 이용자 기기 + 클라우드 백업(동의 시)
일상 기록·병원 방문·진료 기록 이용자 기기 + 클라우드 백업(동의 시)
알러지·기저질환 정보 이용자 기기 + 클라우드 백업(동의 시)
계정 정보 (이메일·이름·사용자 ID) Firebase Authentication
클라우드 백업 암호문 + 암호화 키 Cloudflare 백업 서버 (아시아 태평양 region)

4. 보관되는 데이터 (삭제 후에도 일정 기간 유지)

다음 데이터는 법령 또는 서비스 운영을 위해 제한된 기간 동안 자동 보관 후 자동 삭제됩니다. 이 데이터는 개인을 직접 식별하지 않습니다.

  • 크래시·오류 로그: 수집일로부터 90일 후 자동 삭제 (Firebase Crashlytics)
  • 익명 사용 통계: 수집일로부터 14개월 후 자동 삭제 (Firebase Analytics)
  • 법령상 보관 의무: 관계 법령에서 보관을 요구하는 항목에 한해 해당 기간 동안 보관

5. 처리 결과 안내

  • 앱 내 직접 삭제 — 실행 즉시 적용. 별도 안내 없음.
  • 이메일 요청 삭제 — 접수 확인 메일 발송 후, 처리 완료 시 완료 메일 발송.
  • 삭제 완료 후에는 동일 계정·기기에서도 데이터 복구가 불가능합니다.
